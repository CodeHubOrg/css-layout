Some of the examples show the 'Holy Grail Layout'. Although this layout can be faked using earlier web technologies (floats, inline-block) those techniques come with a lot of caveats and are not covered in this workshop.

'The Holy Grail Layout' can now be achieved easily with newer web technologies (flexbox and grid) without the caveats.
