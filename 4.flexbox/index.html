Flexbox

First proposed as part of the W3C specification in 2009. Currently fully supported by all latest browsers. Partially supported by IE11 and IE10.

Flexbox is the first real tool for web developers that is designed and is intended to help web developers layout there websites. Huzzah!.

The introduction of flexbox heralds what Eric Meyer calls 'The era of intential layout'.

What is it good at?

  • Laying items out over a line either horizontally left to right or right to left or vertically top to bottom or bottom to top.
